# Pattern Library Update v1.1

実記事UATで再現性を確認した共通規則を、8種類のPrimary Patternに横断適用する。

## 共通追加規則
1. **Intent-first title**：主検索意図をタイトル前半で明示し、別テーマやサイト全体の訴求を混在させない。
2. **Answer-first introduction**：導入で結論・解決可能範囲・前提条件を早期提示する。
3. **FAQ complement**：FAQは本文の重複を避け、例外、失敗、選び方、次の疑問を補完する場合のみ追加する。
4. **Internal-link relevance**：候補数の充足より、読者の次の行動との意味的関連性を優先する。
5. **Asset preservation**：比較表、独自体験、PR表記、広告・商品リンク、既存の有用な結論を保護する。
6. **YMYL safety override**：健康・安全上の危険がある場合、PatternよりSafety Gateを優先する。

## 適用方針
これらは新しいPrimary Patternを追加するものではない。既存8パターンのComposer共通規則として適用し、Pattern Manifestの互換性を維持する。
